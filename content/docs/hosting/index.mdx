## Platform Features

Our nodes are tuned specifically to handle performance-heavy game instances with zero overhead.

- **Pelican Control Panel:** Modern, responsive, web-based daemon management that makes handling console inputs effortless.
- **Instant Port Allocation:** Simple networks allow easy setup for specialized plugins like Dynmap, Voice Chat, or Veloctiy proxies.
- **Automated Backups:** Don't lose hours of builds. Schedule automatic system snapshots straight from your schedules tab.
