---

### Examples

```lua title="Create a simple addon"
local addon_manager = require(game:GetService("ReplicatedStorage").AddonManager)
local addon_config = {
  name = "MyAddon",
  description = "A simple addon example.",
  version = "1.0.0",
}


addon_manager:CreateAddon(addon_config, function()
  print("MyAddon has been loaded!")
end)

-- Or just


function myFunction()
  print("MyAddon has been loaded!")
end

addon_manager:CreateAddon(addon_config, myFunction())

```
