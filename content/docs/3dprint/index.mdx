# Welcome!

This documentation hub is designed to help you manage your 3D print orders and configure your custom NFC tags. Whether you are adjusting a redirect URL for a smart tag or troubleshooting a print, you will find all the necessary guides right here.

## Project Highlights

We focus on high-quality prints and seamless digital integration.

- **NFC Integration:** Easily rewrite and manage your embedded NFC tags to point to any URL or action.
- **Precision 3D Printing:** Documentation on material settings, orientation, and post-processing for the best finish.
- **Custom Workflows:** Step-by-step guides for custom requests and assembly instructions.
